# BasicUnits
A converter to help you with basic units.  
Convert quantities from one unit to another.
## Description
### Supported Quantities
- Length  
- Mass  
- Time  
- Temperature
### Supported Units
#### Length
- metre(m)  
- centimetre(cm)  
- kilometre(km)  
- foot(ft)  
- inch(in)  
- mile(mi)
#### Mass
- gram(g)  
- kilogram(kg)  
- tonne(t)  
- pound(lb)  
#### Time
- second(s)  
- minute(min)  
- hour(hr)  
#### Temperature
- Kelvin(K)  
- Celsius(C)  
- Fahrenheit(F)
## Installation
```
npm install basic-units
```
## Usage
```
import {metreTo, gramTo} from 'basic-units'
metreTo("centimetre", 1.35)); // 135
gramTo("kilogram", 1680)); // 1.68
```
